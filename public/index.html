• "/" корневой путь, реализован через GET-запрос, открывает данную страницу

  • "/stats" - возвращает запросы через GET-запрос

  • "/comments" - показывает комментарии через POST-запрос

  • "/users?apiKey=kotiki"- через POST-запрос происходит авторизация

  • "/dbcomments" - показывает комментарии из базы данных через GET-запрос

  • "/dbcomments/:id" - показывает комментарий с конкретным id из базы данных через GET-запрос

  • "/dbcomments" - добавляет комментарий в базу данных через POST-запрос
